In the Stack

The AI space develops fast, with one cornerstone being open datasets available for training. Code from golang-leipzig is part of The Stack (20 Nov 2022):

a 6.4 TB dataset of permissively licensed source code in 358 programming languages.

Five of our repositories are included in the 25.74 GB of source code (in 4,730,461 files) written in Go included in this dataset.

Transparency is a key factor in the development of datasets and derived models. To check, whether your code is included, visit: https://huggingface.co/spaces/bigcode/in-the-stack

May the source be with you!
