Drinkup notes

Meetup #39 took place 2023-12-19 at Cantona Leipzig and these were just some of the talking points.

  • without extra libraries, schema validation in JSON is minimalistic (e.g. json schema); could cue help?
  • malloy is an experimental data modeling language over relational stores
  • direnv is a nice shell helper for 12-factor-style appu
  • time is not just a line, it’s bound (cf. chrony)
  • test your latencies: aws, gcp, or just the internet
  • eventbridge/kubernetes at serverless office hours
  • interesting, dedicated code assistant for ansible: Ansible Lightspeed

Next year will probably bring yet another wave of hyper-focussed LLM-based code comprehension tools (beside hyper-focussed niche, local, custom-trained LLMs for dedicated tasks, in general).

Happy holidays!
