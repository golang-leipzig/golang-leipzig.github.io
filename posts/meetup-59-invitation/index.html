Sandboxed Agents with yoloAI

Hello, 世界!

For meetup #59 on Tuesday Apr 28, 2026 19:00 CET we are thrilled to welcome back Karl Stenerud, who will be presenting one of his recent projects: yoloAI:

Run AI coding agents freely in a disposable sandbox. Review exactly what changed. Apply what you want.

Some people are aware of the security nightmares that probabilisic language models with full system access may have, but you also need to act on it.

Permission fatigue is a real problem. Sandbox escape is a real problem. yoloAI solves it.

The project is written in Go and open source at kstenerud/yoloai, works with various sandbox backends like docker, podman, tart, seabelt - and security modes, using runc, gvisor, kata, firecracker.

After looking at a agent memory upgrade system in #58, we continue our exploration of agentic coding and ai assisted software development, which is a rapidly developing field.
