Rust x Go: Everything is Open Source

Hello, 世界!

We collaborate with Rust Leipzig User Group for a special event on 2026-06-16 19:00 at Basislager Coworking, which we call Everything is Open Source.

Disassembling and decompiling code was once considered a highly complicated task, making obfuscating functionality within propriety software a viable strategy for decades […] LLMs have recently leveled the playing field considerably, also in this domain, breaking a great number of assumptions once believed to be safe.

This meetup is intended as an interactive session: Let’s compile some demo code, C, Go, Rust, Zig, etc., and see how far it can be reversed with contemporary tooling. Bring your laptop, a compiler and your favorite LLM-based assistants. Lightning talks on fun topics around binaries welcome.

We’ll meet in person at Basislager Leipzig, at Peterssteinweg 14, 04107 Leipzig (OSM)
