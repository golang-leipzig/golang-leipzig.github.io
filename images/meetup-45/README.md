Requirements:

* Python 3 and requirements
* FFMPEG
* make

![](anim.webm)

## Build

```shell
$ mkvenv
$ pip install -r requirements.txt
$ make
```

Building the gif and webm took 49s on a i7-8550U 2017 CPU.
